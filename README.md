# Dota 2 Hero Telegram Bot

Бот для получения информации о героях Dota 2: атрибуты, способности, улучшения от Аганима и Осколка.

## Команды
- `/start` – главное меню
- `/help` – справка

Управление через инлайн-кнопки.

## Установка и запуск
1. Клонировать репозиторий
2. Создать виртуальное окружение
3. Установить зависимости: `pip install -r requirements.txt`
4. Создать `.env` с переменной `BOT_TOKEN`
5. Запустить: `python movie_bot.py`

## Источники данных
- OpenDota API
- dotaconstants (кешировано локально)

## Технологии
- Python 3.10+
- python-telegram-bot
- requests
