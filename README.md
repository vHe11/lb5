# Dota 2 Hero Telegram Bot

Бот предоставляет подробную информацию о героях Dota 2: их атрибуты, способности и улучшения от Aghanim's Scepter и Aghanim's Shard.

## Команды и управление
Взаимодействие с ботом происходит через кнопки:
*   `/start` — Главное меню.
*   **Главное меню** -> **«🎮 Выбрать героя»** -> Выберите героя из списка.
*   После выбора героя вам станут доступны кнопки **«📊 Атрибуты»** и **«🪄 Способности + улучшения»**.
*   Навигация по длинному списку героев — с помощью кнопок **«◀ Назад»** и **«Вперёд ▶»**.
